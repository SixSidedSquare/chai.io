= chai.io

A Rails app for Data Visualization. Interface to create reports with SQL queries and data visualization via Graphs, Charts etc.

Its in a very early stage, so things may change.

Also, if you use it don't forget to change the config/initializers/secret_token.rb as a few important features are using keys from there.
